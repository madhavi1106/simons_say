# Simon Says Game 🎮

This is a Simon Says memory game built using HTML, CSS, and JavaScript.

The project was initially developed as part of a course, and I enhanced it by:
- Improving the UI design
- Adding background styling
- Refining button effects
- Understanding and restructuring the game logic

## Features
- Random color sequence generation
- User input validation
- Level tracking
- Game over detection
- Responsive UI

## Tech Used
- HTML
- CSS
- JavaScript (DOM manipulation & event handling)

## Learning Outcome
Focused on understanding how game state, event listeners, and logic flow work together in a real project.
